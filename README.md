## Skeleton aka boilerplate for react and threejs (WebGL) scene based app/game/experiment

### Running the game
1) `npm i` - first install dependencies
2) `npm start` - just start the webpack and dev server

### Features
- gui
- shaders
- scenes
- loading
- saving
- stats
- code splitting
